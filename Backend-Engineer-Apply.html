Backend Engineer

Design and implement scalable backend services for our document processing pipeline and user management systems.

Backend Engineer

Engineering Full-time Remote Friendly
Apply Now

Join our backend engineering team to build the infrastructure that powers document processing for millions of users. You'll design, implement, and scale the services that handle everything from user authentication to complex document transformations.

About the Role

As a Backend Engineer at Photo Flare, you'll be responsible for designing, building, and maintaining the scalable backend services that power our document processing platform. You'll work on everything from high-performance APIs to data processing pipelines, ensuring our systems can handle millions of requests while maintaining reliability and performance.

Our backend stack includes Node.js, Python, PostgreSQL, Redis, and AWS services. You'll have the opportunity to work on challenging distributed systems problems and contribute to our technical direction as we scale our platform.

Our Tech Stack

Node.js Python PostgreSQL Redis AWS Docker Kubernetes GraphQL REST APIs Microservices

Key Responsibilities

  • Design, develop, and maintain scalable backend services and APIs
  • Optimize database performance and query efficiency
  • Implement and maintain data processing pipelines for document operations
  • Build and improve user authentication and authorization systems
  • Monitor system performance and implement improvements
  • Collaborate with frontend engineers to design API contracts
  • Write clean, maintainable code with comprehensive test coverage
  • Participate in code reviews and provide constructive feedback
  • Debug and resolve production issues across multiple services

Qualifications & Requirements

  • 4+ years of professional backend development experience
  • Strong proficiency in Node.js and/or Python
  • Experience with relational databases (PostgreSQL, MySQL) and database design
  • Knowledge of caching systems (Redis, Memcached)
  • Experience with cloud platforms (AWS, GCP, or Azure)
  • Understanding of RESTful API design and best practices
  • Familiarity with containerization (Docker) and orchestration (Kubernetes)
  • Experience with version control systems, preferably Git
  • Strong problem-solving and communication skills
  • Bachelor's degree in Computer Science or equivalent experience

Bonus Points For

  • Experience with document processing libraries or frameworks
  • Knowledge of GraphQL and its implementation
  • Experience with message brokers (RabbitMQ, Kafka)
  • Background in security best practices and implementation
  • Experience with infrastructure as code (Terraform, CloudFormation)
  • Contributions to open source projects
  • Experience with performance monitoring and optimization

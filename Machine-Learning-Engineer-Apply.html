Machine Learning Engineer

Develop AI-powered features for document understanding, image processing, and intelligent automation.

Machine Learning Engineer

Engineering Full-time
Apply Now

Join our AI team to build the next generation of intelligent document processing tools. You'll develop machine learning models that understand, analyze, and transform documents and images, making complex tasks simple for millions of users.

About the Role

As a Machine Learning Engineer at Photo Flare, you'll be responsible for designing, developing, and deploying machine learning models that power our document understanding and image processing features. You'll work on everything from computer vision models that extract text from images to NLP models that understand document structure and content.

Our ML stack includes Python, TensorFlow, PyTorch, and various cloud ML services. You'll have the opportunity to work on challenging problems in document understanding, computer vision, and natural language processing, with your work directly impacting millions of users.

Our ML Tech Stack

Python TensorFlow PyTorch OpenCV Scikit-learn Hugging Face AWS SageMaker Docker Kubernetes MLflow Apache Spark

Key Responsibilities

  • Design, develop, and deploy machine learning models for document understanding
  • Implement computer vision algorithms for image processing and analysis
  • Build and optimize NLP models for text extraction and analysis
  • Collaborate with product teams to define ML requirements and use cases
  • Preprocess and analyze large datasets for model training
  • Implement MLOps practices for model deployment and monitoring
  • Optimize model performance for production environments
  • Stay current with latest developments in ML research
  • Collaborate with software engineers to integrate ML models into products

Qualifications & Requirements

  • 3+ years of experience in machine learning engineering
  • Strong programming skills in Python
  • Experience with deep learning frameworks (TensorFlow, PyTorch)
  • Knowledge of computer vision or natural language processing
  • Experience with ML model deployment and MLOps practices
  • Familiarity with cloud platforms (AWS, GCP, or Azure)
  • Understanding of software engineering best practices
  • Experience with data preprocessing and analysis
  • Strong problem-solving and analytical skills
  • Master's or PhD in Computer Science, ML, or related field preferred

Bonus Points For

  • Experience with document understanding or OCR technologies
  • Publications in ML conferences or journals
  • Experience with distributed training of ML models
  • Knowledge of information retrieval systems
  • Experience with generative models (GANs, VAEs, diffusion models)
  • Background in mathematics or statistics
  • Contributions to open source ML projects

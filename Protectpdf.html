Frequently Asked Questions

How secure is PDF password protection?

Our PDF protection uses industry-standard AES encryption with 128-bit or 256-bit key lengths, providing military-grade security for your documents.

Can I remove PDF protection if I forget the password?

No, for security reasons, we cannot recover or remove passwords from protected PDFs. We recommend storing your passwords securely.

What's the difference between user and owner passwords?

A user password is required to open the PDF. An owner password controls permissions (printing, editing, etc.) and can be different from the user password.

Is there a file size limit for PDF protection?

Free users can protect PDF files up to 10MB. Premium users can protect files up to 100MB (monthly) or 500MB (yearly plan).

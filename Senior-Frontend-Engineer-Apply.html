Senior Frontend Engineer

Build responsive, performant user interfaces for our document processing tools using modern JavaScript frameworks.

Senior Frontend Engineer

Engineering Remote Full-time
Apply Now

Join our team to build the future of document processing with cutting-edge frontend technologies. We're looking for an experienced engineer who can architect and implement responsive, performant user interfaces that delight our millions of users.

About the Role

As a Senior Frontend Engineer at Photo Flare, you'll be responsible for designing, building, and maintaining the user interfaces for our document processing tools. You'll work closely with product managers, designers, and backend engineers to create seamless user experiences that handle complex document operations with ease.

Our frontend stack includes React, TypeScript, and modern CSS, and we're constantly exploring new technologies to improve our user experience. You'll have the opportunity to influence our technical direction and mentor other engineers on the team.

Key Responsibilities

  • Design and implement responsive, accessible user interfaces using React and TypeScript
  • Optimize frontend performance to ensure fast loading times and smooth interactions
  • Collaborate with UX/UI designers to translate mockups into functional components
  • Write clean, maintainable code with comprehensive test coverage
  • Mentor junior engineers and contribute to team knowledge sharing
  • Participate in code reviews and provide constructive feedback
  • Work closely with backend engineers to design API contracts
  • Stay current with frontend technologies and industry trends

Qualifications & Requirements

  • 5+ years of professional frontend development experience
  • Expert knowledge of JavaScript/TypeScript, HTML5, and CSS3
  • Extensive experience with React and its ecosystem (Redux, React Router, etc.)
  • Strong understanding of responsive design principles and cross-browser compatibility
  • Experience with frontend testing frameworks (Jest, React Testing Library, Cypress)
  • Familiarity with build tools and package managers (Webpack, Vite, npm/yarn)
  • Experience with version control systems, preferably Git
  • Excellent problem-solving and communication skills
  • Bachelor's degree in Computer Science or equivalent experience

Bonus Points For

  • Experience with document processing libraries (PDF.js, etc.)
  • Knowledge of WebAssembly and its applications
  • Experience with data visualization libraries (D3.js, Chart.js, etc.)
  • Contributions to open source projects
  • Experience with CI/CD pipelines and DevOps practices
  • Familiarity with cloud platforms (AWS, Azure, GCP)
Estimated Salary Range
$120,000 - $160,000
+ equity + benefits. Actual compensation will be based on experience and location.

Location

Remote (US/Canada) or San Francisco, CA

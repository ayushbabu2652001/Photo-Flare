Frequently Asked Questions

What is selective sync and why is it useful?
Selective sync allows you to choose specific folders to sync between cloud services, rather than syncing everything. This is useful for saving bandwidth, storage space, and time by excluding unnecessary files like system folders, temporary files, or large archives that you don't need to sync across all your devices.
How does real-time syncing work?
Real-time syncing uses advanced file monitoring technology to detect changes immediately when they occur. When you add, modify, or delete a file in any connected cloud storage, Photo Flare detects the change and synchronizes it across all other connected services within seconds, without waiting for scheduled sync intervals.
Can I set different sync settings for different cloud services?
Yes, with Photo Flare Premium you can customize sync settings individually for each cloud service connection. For example, you could set real-time syncing for your primary cloud but only hourly syncing for a backup cloud, or use selective sync for one service while syncing everything from another.
How does bandwidth control help with sync operations?
Bandwidth control allows you to limit how much network capacity sync operations can use. This prevents large sync jobs from slowing down your other internet activities like video streaming, online gaming, or video calls. You can set different limits for upload and download speeds, and even schedule different limits for different times of day.
